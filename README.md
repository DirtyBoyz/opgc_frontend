# opgc_frontend

Over Programmed Good Coding 프론트엔드 레포지토리 <br>
실 서버 구축 전까지, [vercel을 이용해 배포](https://opgc-frontend.vercel.app/)하여 상태를 공유한다.

## 기술 스택

- antd
- redux (사용예정)
- next
- react
- typescript

## 로컬 서버 구동

```bash
> npm run start ## 일반
or
> npm run dev ## hot-reload 시
```

## 빌드

```bash
> npm run build
```
